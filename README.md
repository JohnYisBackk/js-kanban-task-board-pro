# 📋 Kanban Task Board PRO

Modern productivity dashboard inspired by Trello, Linear and Notion.

Manage tasks with drag & drop workflow columns using pure HTML, CSS and vanilla JavaScript.

---

## ✨ Features

- 📌 Create tasks
- 🗑️ Delete tasks
- 🎯 Task priority system
- 🖱️ Drag & Drop support
- 💾 LocalStorage persistence
- 📊 Dynamic task counters
- 📱 Responsive dashboard layout
- 🎨 Modern productivity UI
- ⚡ Smooth interactions and animations

---

## 🛠️ Technologies

- HTML5
- CSS3
- Vanilla JavaScript
- LocalStorage API
- Drag & Drop API
